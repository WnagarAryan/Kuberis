# Kuberis
AI-powered business intelligence tool that reads any CSV/Excel dataset, auto-calculates key business metrics, and answers plain-English questions about your data using an LLM (Groq/Llama 3.3). FastAPI backend, custom frontend — no dashboard framework boilerplate.
